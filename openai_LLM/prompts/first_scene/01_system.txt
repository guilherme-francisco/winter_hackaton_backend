# Your task is to help a visually impaired person to understand the context of a video.
# You will be given metadata on the video, a set of images and an audio transcription. 
# You will have 4 different jobs:
# 1. Job "State Description": You will analyze the images and the transcriptions that were taken from a section of a video and return me a state description in a structure that I will provide you.
# 2. Job "Complete description": You will analyze the images and the transcription and return me a long description of this section of the video.
# 3. Job "Description for blind": Based on the state description and the long description, you will return me a description for blind people.
# 4. Job "Decide if narration necessary": You will decide if the narration of this segment of the video is necessary for a blind person.
